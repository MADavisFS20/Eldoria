# Eldoria: A Text-Based RPG

Welcome to Eldoria, a text-based role-playing game for the command-line interface (CLI)!
Embark on an adventure in a medieval world, explore various locations, fight monsters, complete quests, and develop your character.

## Features

*   **Character Progression:** Level up, gain experience, and improve your stats.
*   **Combat System:** Turn-based combat against various enemies.
*   **Inventory & Equipment:** Collect items, equip weapons and armor, and use potions.
*   **Exploration:** Navigate through interconnected locations in a medieval world.
*   **Quests:** Undertake and complete quests from NPCs.
*   **Save/Load Game:** Save your progress and continue your adventure later.
*   **Shop System:** Buy and sell items in town to manage your inventory and gold.
*   **Abbreviated Commands:** Use the first letter of most commands (e.g., 'm' for 'move', 'i' for 'inventory') for quicker input.

## How to Play

### 1. Setup

To run this game, you need Python 3 installed on your system.

1.  **Save the files:** Create a directory named `rpg_game` and place all the provided Python files (`game.py`, `player.py`, `world.py`, `combat.py`, `items.py`, `quests.py`, `utils.py`, `shop.py`) inside it.
2.  **Create `main.py`:** Place the `main.py` file in the *parent directory* of `rpg_game`.
3.  **Create `__init__.py`:** Create an empty file named `__init__.py` inside the `rpg_game` directory.
4.  **Navigate to the project root:** Open your terminal or command prompt and navigate to the directory containing `main.py`.
