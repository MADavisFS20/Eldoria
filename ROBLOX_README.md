# Eldoria: Roblox Edition

A visual RPG game for Roblox based on the original text-based Eldoria game. Features combat, character progression, equipment systems, and premium monetization.

## Project Structure

### Core Modules
- `Roblox/StarterPlayer/` - Player-side local scripts and UIs
- `Roblox/ServerScriptService/` - Server-side game logic
- `Roblox/ReplicatedStorage/Modules/` - Shared game modules

### Key Features
1. **Character System** - Classes, attributes, stats progression
2. **Combat System** - Turn-based combat with spells and abilities
3. **Equipment System** - Weapons, armor, accessories with stat bonuses
4. **Progression** - Leveling, perks, attribute allocation
5. **Shop System** - NPC shops for items and premium Robux shop
6. **Premium Monetization** - Exclusive cosmetics, battle pass, XP boosters

## Installation

1. Open Roblox Studio
2. Create a new place
3. Copy the contents of the `Roblox/` folder into your game
4. Configure MarketplaceService IDs in `ServerScriptService/MarketplaceManager`
5. Publish and deploy
